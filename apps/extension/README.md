# Chrome / Edge MV3 extension (deferred to v0.5)

The clipping extension is **deferred** until the web app is stable. See `docs/superpowers/specs/2026-06-28-snowstep-doc-design.md` module M2 for the original scope.

The M2 spec requires:

- One-click clip from any page
- Editable extracted text + selection support
- Drop into a category (via the existing `category` frontmatter)

A follow-up plan will be created once the web app is in production. Until then, the workaround is: copy/paste the URL into `/search` or save the markdown directly under `apps/web/src/content/clipped/`.
